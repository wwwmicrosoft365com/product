Using The Control Panel:

  • First of all, you need to click on the Start button.
  • The start button can be found in the bottom left screen of the system.
  • Furthermore, select the Control Panel from the menu.
  • Now, you will need to select Program & Features option to proceed further.
  • After that, select the Program tab and then click on the Uninstall a program option.
  • Now, from the list, you need to select the program which you want to remove from your system.
  • Wait to finish the removal of the program from the system.
  • After that, click on the Restart Now, to save the changes you have made.

Now, you need to wait until the system restarts again and then you can follow the procedure for Office Reinstall. The re-installation process is very easy and requires no technical knowledge.
